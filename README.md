NOTIFIKASI TELEGRAM OTOMATIS
Front End Side 

Cara Menggunakan:
1. Buat file HTML. Isikan dengan isi file notifikasiotomatis.html.
2. Masukkan token BOT dan ChatID ke dalam kode HTML.
3. Jalankan kode HTML melalui browser. 
Atur pesan dan waktu pengiriman otomatis yang tertera di dalam kode.
4. Notifikasi aktif selama browser
menjalankan HTML tersebut.
